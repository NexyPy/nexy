# CLI Reference

The Nexy CLI (`nx`) is a powerful tool for managing your development workflow.

## Commands

### `nx init`

Initializes a new Nexy project. Follow the interactive assistant to choose your frontend framework, enable Tailwind CSS, and set up your project structure.

```bash
nx init my-app
```

### `nx dev`

Starts the Nexy development server. This command will start the FastAPI backend and the Vite orchestrator in parallel, enabling hot-reload (HMR).

```bash
nx dev
```

### `nx build`

Compiles your assets and prepares your project for production deployment. This command optimizes your code and assets for better performance.

```bash
nx build
```

### `nx start`

Starts the Nexy production server. This command will run your application in production mode without the overhead of the development server.

```bash
nx start
```

### `nx add`

Adds new components or features to your Nexy project.

```bash
nx add auth
```

### `nx deploy`

Deploys your Nexy application to a cloud provider.

```bash
nx deploy
```

---

**Tip**: Use `nx --help` for a full list of commands and options.
