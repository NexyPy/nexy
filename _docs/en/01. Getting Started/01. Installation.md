# Installation

Nexy is easy to install and get started with.

## Quick Start

The fastest way to get started is by using the `nx init` command.

### 1. Install Nexy via pip

```bash
pip install nexy
```

### 2. Initialize a project

```bash
nx init my-app
```

The interactive assistant will guide you through:
- Choosing your preferred frontend framework (React, Vue, Svelte, or Solid.js).
- Enabling or disabling Tailwind CSS.
- Setting up the project structure.

## Prerequisites

- **Python 3.10+**
- **Node.js 18+** (Required only if using JS frontend frameworks)

## Running in Development

Once your project is initialized, start the development server:

```bash
nx dev
```

This command will:
- Start the **FastAPI** backend server.
- Start the **Vite** frontend orchestrator in parallel.
- Enable **Hot Module Replacement (HMR)** for a seamless development experience.

---

**Tip**: Use the `--help` flag with any `nx` command to see available options.
