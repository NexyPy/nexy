# Introduction

Nexy is a full-stack Python meta-framework designed for building modern web applications. It unifies **FastAPI** for backend logic and **Vite** for interactive frontend development, providing a single ecosystem for developers.

## Key Principles

- **Unified Type-Safety**: Use Pydantic to define data models once and use them across the entire stack.
- **Server-First (SSR)**: Applications are rendered on the server by default for optimal performance and SEO.
- **Selective Interactivity**: JavaScript is treated as an optional integration, activated only when rich client-side interactivity is required.
- **Architectural Flexibility**: Choose between file-based routing for content-heavy pages or module-based routing for enterprise APIs.
