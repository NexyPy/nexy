# Routes Router: Pages, APIs & WebSockets

The **Routes Router** (File-Based Routing) is Nexy's default engine. It uses the structure of your `src/routes/` directory to automatically define your application's entry points, whether they are HTML pages, REST APIs, or WebSocket connections.

## 1. Page Rendering (UI)

`.nexy` and `.mdx` files are automatically transformed into HTML routes.

- `src/routes/index.nexy` -> `/`
- `src/routes/blog/[slug].mdx` -> `/blog/{slug}`

### Layouts
A `layout.nexy` file in a directory automatically applies to all routes in that directory and its subdirectories.

## 2. API Endpoints (.py)

To create pure API endpoints (JSON), use `.py` files within `src/routes/`. Nexy identifies functions whose names match HTTP verbs in **uppercase**.

```python
# src/routes/api/users.py
from pydantic import BaseModel

class User(BaseModel):
    id: int
    name: str

# GET /api/users endpoint
async def GET():
    return [{"id": 1, "name": "Alice"}]

# POST /api/users endpoint
async def POST(user: User):
    return {"status": "created", "user": user}
```

## 3. WebSocket Support

The Routes Router also supports WebSockets via a function named `SOCKET` (in uppercase) within a `.py` file.

```python
# src/routes/chat.py
from fastapi import WebSocket

async def SOCKET(websocket: WebSocket):
    await websocket.accept()
    await websocket.send_text("Connected to Routes Router WebSocket!")
    await websocket.close()
```

## 4. Dependency Management

Nexy provides a hierarchical dependency system for the Routes Router. Create a `dependencies.py` file in any subdirectory of `src/routes/`.

```python
# src/routes/admin/dependencies.py
from fastapi import Header, HTTPException

async def verify_token(x_token: str = Header(...)):
    if x_token != "fake-super-secret-token":
        raise HTTPException(status_code=400, detail="X-Token header invalid")

# All routes in /admin/ and its subfolders
# will use these dependencies.
dependencies = [verify_token]
```

### Dependency Features:
- **Inheritance**: Dependencies defined at the root of `src/routes/` apply to the entire application.
- **Accumulation**: A child folder adds its own dependencies to those of its parents.
- **FastAPI Native**: Nexy uses `fastapi.Depends` internally, allowing you to use all of FastAPI's features.

---

**Tip**: Use `.py` files for lightweight endpoints and the **Modular Router** for complex services requiring advanced class-based dependency injection.
