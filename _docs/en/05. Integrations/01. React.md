# Integrations: React, Vue, Svelte, Solid.js

Nexy is framework-agnostic. You can use any of the major JavaScript frontend frameworks within your server-rendered pages.

## How it Works

Nexy uses **Vite** as an orchestrator to bundle and hydrate client-side components.

- **Automatic Bundling**: When you import a `.tsx`, `.vue`, `.svelte`, or `.solid` component into a `.nexy` file, Nexy triggers Vite to create the client-side bundle.
- **Hydration**: Client components are rendered to HTML on the server and then hydrated on the client to make them interactive.
- **Zero Configuration**: Nexy provides a pre-configured Vite setup for each framework.

## Supported Frameworks

- **React**: Full support for React components and hooks.
- **Vue**: Support for SFCs (Single File Components).
- **Svelte**: Support for Svelte components.
- **Solid.js**: Support for Solid.js components.

### Example React Component

```tsx
// src/components/Counter.tsx
import { useState } from 'react';

export default function Counter() {
    const [count, setCount] = useState(0);
    return (
        <button onClick={() => setCount(count + 1)} className="px-4 py-2 bg-blue-500 text-white rounded">
            Count: {count}
        </button>
    );
}
```

### Usage in .nexy

```python
---
from "@/components/Counter.tsx" import Counter
---
<Counter />
```

---

**Tip**: Choose the framework that best fits your team's expertise. Nexy's orchestration ensures that you don't have to worry about the underlying build process.
