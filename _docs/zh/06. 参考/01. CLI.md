# CLI 参考

Nexy CLI (`nx`) 是管理开发流程的强大工具。

## 命令

### `nx init`

初始化一个新的 Nexy 项目。按照交互式助手选择您的前端框架、激活 Tailwind CSS 并配置项目结构。

```bash
nx init my-app
```

### `nx dev`

启动 Nexy 开发服务器。该命令并行启动 FastAPI 后端和 Vite 编排器，并激活热模块替换 (HMR)。

```bash
nx dev
```

### `nx build`

编译您的资源并为生产环境部署准备项目。该命令优化您的代码和资源以获得更好的性能。

```bash
nx build
```

### `nx start`

启动 Nexy 生产服务器。该命令在生产模式下运行您的应用程序，没有开发服务器的开销。

```bash
nx start
```

### `nx add`

向您的 Nexy 项目添加新组件或功能。

```bash
nx add auth
```

### `nx deploy`

将您的 Nexy 应用程序部署到云提供商。

```bash
nx deploy
```

---

**建议**：使用 `nx --help` 获取命令和选项的完整列表。
