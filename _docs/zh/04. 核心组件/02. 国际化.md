# 国际化 (i18n)

Nexy 集成了一个简单且高性能的国际化系统，使全球用户都能访问您的应用程序。

## 1. 文件结构

翻译文件以 JSON 格式存储在 `locales/` 目录（或您配置中定义的目录）中。

```text
my-project/
├── locales/
│   ├── fr.json
│   └── en.json
└── src/
    └── routes/
        └── index.nexy
```

### `fr.json` 文件示例
```json
{
  "welcome": "欢迎来到 Nexy",
  "description": "Python 全栈框架"
}
```

## 2. 在 .nexy 文件中使用

您可以直接在模板或 Python 逻辑中使用 `t()` (translate) 函数。

```html
---
from nexy import t
---
<section>
    <h1>{{ t("welcome") }}</h1>
    <p>{{ t("description") }}</p>
</section>
```

## 3. 语言检测

Nexy 通过以下方式自动检测用户语言：
1.  **URL 参数**：`?lang=fr`
2.  **HTTP Header**：`Accept-Language`
3.  **Cookie**：`nexy_locale`

## 4. 动态翻译（变量）

您可以将变量传递给字符串。

### JSON 文件
```json
{
  "greeting": "你好 {name} ！"
}
```

### 用法
```python
{{ t("greeting", name="Alice") }}
```

## 5. 配置

您可以在 `nexyconfig.py` 中定义默认语言。

```python
from nexy.core.models import NexyConfigModel

class NexyConfig(NexyConfigModel):
    useLocaleDefault = "fr"
    useLocales = ["fr", "en"]
```

---

**建议**：从项目开始就使用国际化，以便于您将来扩展应用程序。
