# 介绍

Nexy 是一个专为开发现代 Web 应用程序设计的 Python 全栈元框架。它将用于后端逻辑的 **FastAPI** 与用于交互式前端开发的 **Vite** 统一起来，为开发者提供了一个完整的生态系统。

## 核心原则

- **统一的类型安全 (Pydantic)**：只需定义一次数据模型，即可在整个堆栈中确保类型验证和一致性。
- **服务器优先 (SSR)**：应用程序默认在服务器端渲染，以获得最佳的 SEO 和性能。
- **选择性交互**：JavaScript 被视为可选集成，仅在需要丰富的客户端交互时才激活。
- **灵活的架构**：在基于文件的路由（适用于内容丰富的页面）和基于模块的路由（适用于企业级 API）之间进行选择。
