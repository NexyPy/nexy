# Modular Router (App Router)

**Modular Router**（位于 `src/app/`）专为企业级架构设计。它基于**控制器**、**模块**和**依赖注入**系统。

与其他框架不同，Nexy 使用基于类方法的方法来定义路由，而不是像 `@Get` 这样的每种方法都要用的装饰器。

## 1. 控制器 (Controllers)

**控制器**是一个使用 `@Controller()` 装饰的类。路由由类中对应于 HTTP 动词（`get`、`post`、`put`、`delete` 等）的**方法名称**定义。

```python
# src/app/blog/blog_controller.py
from nexy.decorators import Controller

@Controller("/blog")
class BlogController:
    def __init__(self, service: BlogService):
        self.service = service

    # 定义 GET /blog 路由
    async def get(self):
        return await self.service.get_all()

    # 定义 POST /blog 路由
    async def post(self, data: dict):
        return await self.service.create(data)
```

### 控制器关键点：
- **方法名称**：使用 `get`、`post`、`put`、`delete`、`patch` 来定义 HTTP 端点。
- **WebSocket**：使用名为 `socket` 的方法来定义 WebSocket 入口点。
- **路由参数**：方法参数会自动绑定到 URL 参数或请求体 (FastAPI)。
- **自动注入**：如果服务被标记为 `@Injectable()`，传递给构造函数 `__init__` 的服务将被自动注入。

## 2. WebSocket 支持

Nexy 通过 Modular Router 原生集成了 WebSocket。只需在您的控制器中定义一个名为 `socket` 的方法。

```python
from nexy.decorators import Controller
from fastapi import WebSocket

@Controller("/ws")
class ChatController:
    async def socket(self, websocket: WebSocket):
        await websocket.accept()
        while True:
            data = await websocket.receive_text()
            await websocket.send_text(f"收到消息: {data}")
```

## 2. 模块 (Modules)

**模块**是一个使用 `@Module()` 装饰的类，它组织了控制器、服务（providers）和其他模块。

```python
# src/app/blog/blog_module.py
from nexy.decorators import Module
from .blog_controller import BlogController
from .blog_service import BlogService

@Module(prefix="/v1")
class BlogModule:
    controllers = [BlogController]
    providers = [BlogService]
```

### 模块属性：
- **controllers**：要注册的控制器类列表。
- **providers**：要使之可注入的服务类列表。
- **imports**：要包含的其他模块（路由器）列表。

## 3. 依赖注入 (DI)

要使服务可注入，请使用 `@Injectable()` 装饰器。

```python
# src/app/blog/blog_service.py
from nexy.decorators import Injectable

@Injectable()
class BlogService:
    async def get_all(self):
        return [{"id": 1, "title": "Hello Nexy"}]
```

## 4. 路由元数据（可选）

如果您需要自定义 OpenAPI 元数据（标签、摘要等），请使用 `UseRoute` 装饰器。

```python
from nexy.decorators import Controller, UseRoute

@Controller("/users")
class UserController:
    @UseRoute(summary="用户列表", tags=["Admin"])
    def get(self):
        return []
```
