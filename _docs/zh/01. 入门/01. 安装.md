# 安装

Nexy 易于安装和上手。

## 快速入门

最快的方法是使用 `nx init` 命令。

### 1. 通过 pip 安装 Nexy

```bash
pip install nexy
```

### 2. 初始化项目

```bash
nx init my-app
```

交互式助手将引导您完成：
- 选择您喜欢的各种前端框架（React、Vue、Svelte 或 Solid.js）。
- 激活或停用 Tailwind CSS。
- 配置项目结构。

## 前提条件

- **Python 3.10+**
- **Node.js 18+**（仅当您使用前端 JS 框架时才需要）

## 在开发模式下运行

项目初始化后，启动开发服务器：

```bash
nx dev
```

该命令将：
- 启动 **FastAPI** 后端服务器。
- 同时启动 **Vite** 前端编排器。
- 启用 **热模块替换 (HMR)** 以获得流畅的开发体验。

---

**建议**：使用任何 `nx` 命令时带有 `--help` 标志以查看可用选项。
