# 集成：React, Vue, Svelte, Solid.js

Nexy 与前端框架无关。您可以在服务器渲染的页面中使用任何主流的 JavaScript 框架。

## 工作原理

Nexy 使用 **Vite** 作为编排器来编译和在客户端激活组件。

- **通过 Vite 进行编排**：要使用这些框架，您必须通过 `nexyconfig.py` 中的 `useVite: True` 激活 Vite 集成。
- **激活 (Hydration)**：客户端组件在服务器上渲染为 HTML (SSR)，然后通过 Vite 基础设施在客户端上激活以实现交互。
- **零配置**：Nexy 为每个受支持的框架提供预先建立的 Vite 配置。

## 受支持的框架

- **React**：对 React 组件和 Hook 的完整支持。
- **Vue**：支持单文件组件 (SFC)。
- **Svelte**：支持 Svelte 组件。
- **Solid.js**：支持 Solid.js 组件。

### React 组件示例

```tsx
// src/components/Counter.tsx
import { useState } from 'react';

export default function Counter() {
    const [count, setCount] = useState(0);
    return (
        <button onClick={() => setCount(count + 1)} className="px-4 py-2 bg-blue-500 text-white rounded">
            计数器：{count}
        </button>
    );
}
```

### 在 .nexy 文件中使用

```python
---
from "@/components/Counter.tsx" import Counter
---
<Counter />
```

---

**建议**：选择最适合您团队专业知识的框架。Nexy 的编排确保您不必担心底层的构建过程。
