# Référence CLI

La CLI de Nexy (`nx`) est un outil puissant pour gérer votre flux de développement.

## Commandes

### `nx init`

Initialise un nouveau projet Nexy. Suivez l'assistant interactif pour choisir votre framework frontend, activer Tailwind CSS et configurer la structure de votre projet.

```bash
nx init my-app
```

### `nx dev`

Démarre le serveur de développement Nexy. Cette commande démarre le backend FastAPI et l'orchestrateur Vite en parallèle, activant le rechargement à chaud (HMR).

```bash
nx dev
```

### `nx build`

Compile vos ressources et prépare votre projet pour le déploiement en production. Cette commande optimise votre code et vos ressources pour de meilleures performances.

```bash
nx build
```

### `nx start`

Démarre le serveur de production Nexy. Cette commande exécute votre application en mode production sans la surcharge du serveur de développement.

```bash
nx start
```

### `nx add`

Ajoute de nouveaux composants ou fonctionnalités à votre projet Nexy.

```bash
nx add auth
```

### `nx deploy`

Déploie votre application Nexy chez un fournisseur cloud.

```bash
nx deploy
```

---

**Conseil** : Utilisez `nx --help` pour obtenir une liste complète des commandes et options.
