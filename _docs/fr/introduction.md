# Introduction

Nexy est un méta-framework full-stack Python conçu pour le développement d'applications web modernes. Il unifie **FastAPI** pour la logique backend et **Vite** pour la gestion réactive du frontend, offrant une solution complète pour les développeurs.

## Principes Clés

- **Type-Safety Unifié (Pydantic)** : Définissez vos modèles de données une seule fois. Nexy assure la validation et la cohérence des types sur toute la stack.
- **Serveur-First (SSR)** : Les applications sont rendues côté serveur par défaut pour un SEO et des performances optimaux.
- **Interactivité Sélective** : Le JavaScript est traité comme une intégration optionnelle, activée uniquement lorsque l'interactivité riche du client est requise.
- **Flexibilité Architecturale** : Choisissez entre le routage par fichiers pour le contenu ou le routage par modules pour les APIs d'entreprise.
