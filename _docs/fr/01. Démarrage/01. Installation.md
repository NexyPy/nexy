# Installation

Nexy est facile à installer et à prendre en main.

## Démarrage Rapide

Le moyen le plus rapide de commencer est d'utiliser la commande `nx init`.

### 1. Installer Nexy via pip

```bash
pip install nexy
```

### 2. Initialiser un projet

```bash
nx init my-app
```

L'assistant interactif vous guidera à travers :
- Le choix de votre framework frontend préféré (React, Vue, Svelte ou Solid.js).
- L'activation ou la désactivation de Tailwind CSS.
- La configuration de la structure du projet.

## Prérequis

- **Python 3.10+**
- **Node.js 18+** (Requis uniquement si vous utilisez des frameworks JS frontend)

## Exécution en Développement

Une fois votre projet initialisé, démarrez le serveur de développement :

```bash
nx dev
```

Cette commande va :
- Démarrer le serveur backend **FastAPI**.
- Démarrer l'orchestrateur frontend **Vite** en parallèle.
- Activer le **Hot Module Replacement (HMR)** pour une expérience de développement fluide.

---

**Conseil** : Utilisez le flag `--help` avec n'importe quelle commande `nx` pour voir les options disponibles.
