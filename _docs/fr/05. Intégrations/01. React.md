# Intégrations : React, Vue, Svelte, Solid.js

Nexy est agnostique vis-à-vis du framework frontend. Vous pouvez utiliser n'importe quel framework JavaScript majeur dans vos pages rendues par le serveur.

## Comment ça fonctionne

Nexy utilise **Vite** comme orchestrateur pour compiler et hydrater les composants côté client.

- **Orchestration via Vite** : Pour utiliser ces frameworks, vous devez activer l'intégration Vite dans `nexyconfig.py` via `useVite: True`.
- **Hydratation** : Les composants client sont rendus en HTML sur le serveur (SSR) puis hydratés sur le client pour devenir interactifs via l'infrastructure Vite.
- **Zéro Configuration** : Nexy fournit une configuration Vite pré-établie pour chaque framework supporté.

## Frameworks Supportés

- **React** : Support complet pour les composants et les hooks React.
- **Vue** : Support pour les SFC (Single File Components).
- **Svelte** : Support pour les composants Svelte.
- **Solid.js** : Support pour les composants Solid.js.

### Exemple de Composant React

```tsx
// src/components/Counter.tsx
import { useState } from 'react';

export default function Counter() {
    const [count, setCount] = useState(0);
    return (
        <button onClick={() => setCount(count + 1)} className="px-4 py-2 bg-blue-500 text-white rounded">
            Compteur : {count}
        </button>
    );
}
```

### Utilisation dans .nexy

```python
---
from "@/components/Counter.tsx" import Counter
---
<Counter />
```

---

**Conseil** : Choisissez le framework qui correspond le mieux à l'expertise de votre équipe. L'orchestration de Nexy garantit que vous n'avez pas à vous soucier du processus de build sous-jacent.
