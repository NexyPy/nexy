# एकीकरण: React, Vue, Svelte, Solid.js

Nexy फ्रंटएंड फ़्रेमवर्क से स्वतंत्र है। आप अपने सर्वर-रेंडर किए गए पृष्ठों में किसी भी प्रमुख JavaScript फ़्रेमवर्क का उपयोग कर सकते हैं।

## यह कैसे काम करता है

Nexy क्लाइंट-साइड पर घटकों को कंपाइल और सक्रिय (hydrate) करने के लिए ऑर्केस्ट्रेटर के रूप में **Vite** का उपयोग करता है।

- **Vite के माध्यम से ऑर्केस्ट्रेशन**: इन फ़्रेमवर्क का उपयोग करने के लिए, आपको `nexyconfig.py` में `useVite: True` के माध्यम से Vite एकीकरण को सक्रिय करना होगा।
- **सक्रियण (Hydration)**: क्लाइंट घटकों को सर्वर (SSR) पर HTML में रेंडर किया जाता है और फिर Vite इंफ्रास्ट्रक्चर के माध्यम से क्लाइंट पर इंटरैक्टिव बनने के लिए सक्रिय किया जाता है।
- **ज़ीरो कॉन्फ़िगरेशन**: Nexy प्रत्येक समर्थित फ़्रेमवर्क के लिए पहले से स्थापित Vite कॉन्फ़िगरेशन प्रदान करता है।

## समर्थित फ़्रेमवर्क

- **React**: React घटकों और हुक के लिए पूर्ण समर्थन।
- **Vue**: SFC (Single File Components) के लिए समर्थन।
- **Svelte**: Svelte घटकों के लिए समर्थन।
- **Solid.js**: Solid.js घटकों के लिए समर्थन।

### React घटक का उदाहरण

```tsx
// src/components/Counter.tsx
import { useState } from 'react';

export default function Counter() {
    const [count, setCount] = useState(0);
    return (
        <button onClick={() => setCount(count + 1)} className="px-4 py-2 bg-blue-500 text-white rounded">
            काउंटर: {count}
        </button>
    );
}
```

### .nexy फ़ाइल में उपयोग

```python
---
from "@/components/Counter.tsx" import Counter
---
<Counter />
```
