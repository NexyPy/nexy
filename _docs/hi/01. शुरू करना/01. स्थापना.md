# स्थापना

Nexy को स्थापित करना और शुरू करना आसान है।

## त्वरित शुरुआत

शुरू करने का सबसे तेज़ तरीका `nx init` कमांड का उपयोग करना है।

### 1. pip के माध्यम से Nexy स्थापित करें

```bash
pip install nexy
```

### 2. एक प्रोजेक्ट शुरू करें

```bash
nx init my-app
```

इंटरैक्टिव सहायक आपको इसके माध्यम से मार्गदर्शन करेगा:
- अपने पसंदीदा फ्रंटएंड फ़्रेमवर्क (React, Vue, Svelte या Solid.js) का चयन करना।
- Tailwind CSS को सक्रिय या निष्क्रिय करना।
- प्रोजेक्ट संरचना को कॉन्फ़िगर करना।

## पूर्वापेक्षाएँ

- **Python 3.10+**
- **Node.js 18+** (केवल तभी आवश्यक है जब आप फ्रंटएंड JS फ़्रेमवर्क का उपयोग करते हैं)

## विकास मोड में चलाना

एक बार जब आपका प्रोजेक्ट शुरू हो जाए, तो विकास सर्वर शुरू करें:

```bash
nx dev
```

यह कमांड:
- **FastAPI** बैकएंड सर्वर शुरू करेगा।
- साथ ही **Vite** फ्रंटएंड ऑर्केस्ट्रेटर शुरू करेगा।
- सुचारू विकास अनुभव के लिए **हॉट मॉड्यूल रिप्लेसमेंट (HMR)** सक्रिय करेगा।

---

**सलाह**: उपलब्ध विकल्पों को देखने के लिए किसी भी `nx` कमांड के साथ `--help` फ़्लैग का उपयोग करें।
