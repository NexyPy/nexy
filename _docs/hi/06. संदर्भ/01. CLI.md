# CLI संदर्भ

Nexy CLI (`nx`) आपके विकास प्रवाह को प्रबंधित करने के लिए एक शक्तिशाली उपकरण है।

## कमांड

### `nx init`

एक नया Nexy प्रोजेक्ट शुरू करता है। अपने फ्रंटएंड फ़्रेमवर्क को चुनने, Tailwind CSS को सक्रिय करने और अपने प्रोजेक्ट की संरचना को कॉन्फ़िगर करने के लिए इंटरैक्टिव सहायक का पालन करें।

```bash
nx init my-app
```

### `nx dev`

Nexy विकास सर्वर शुरू करता है। यह कमांड FastAPI बैकएंड और Vite ऑर्केस्ट्रेटर को समानांतर में शुरू करती है, जिससे हॉट मॉड्यूल रिप्लेसमेंट (HMR) सक्रिय हो जाता है।

```bash
nx dev
```

### `nx build`

आपके संसाधनों को कंपाइल करता है और उत्पादन में तैनाती के लिए आपके प्रोजेक्ट को तैयार करता है। यह कमांड बेहतर प्रदर्शन के लिए आपके कोड और संसाधनों को अनुकूलित करती है।

```bash
nx build
```

### `nx start`

Nexy उत्पादन सर्वर शुरू करता है। यह कमांड विकास सर्वर के ओवरहेड के बिना आपके एप्लिकेशन को उत्पादन मोड में चलाती है।

```bash
nx start
```

### `nx add`

आपके Nexy प्रोजेक्ट में नए घटक या सुविधाएँ जोड़ता है।

```bash
nx add auth
```

### `nx deploy`

आपके Nexy एप्लिकेशन को क्लाउड प्रदाता पर तैनात करता है।

```bash
nx deploy
```

---

**सलाह**: कमांड और विकल्पों की पूरी सूची प्राप्त करने के लिए `nx --help` का उपयोग करें।
