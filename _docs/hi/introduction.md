# परिचय

Nexy एक आधुनिक वेब एप्लिकेशन बनाने के लिए डिज़ाइन किया गया एक Python फुल-स्टैक मेटा-फ़्रेमवर्क है। यह बैकएंड लॉजिक के लिए **FastAPI** और इंटरेक्टिव फ्रंटएंड डेवलपमेंट के लिए **Vite** को एकीकृत करता है, जो डेवलपर्स के लिए एक एकल पारिस्थितिकी तंत्र प्रदान करता है।

## मुख्य सिद्धांत

- **एकीकृत प्रकार-सुरक्षा (Pydantic)**: डेटा मॉडल को एक बार परिभाषित करें और उन्हें पूरे स्टैक में प्रकार सत्यापन और स्थिरता सुनिश्चित करने के लिए उपयोग करें।
- **सर्वर-प्रथम (SSR)**: इष्टतम प्रदर्शन और SEO के लिए एप्लिकेशन डिफ़ॉल्ट रूप से सर्वर-साइड पर रेंडर किए जाते हैं।
- **चयनात्मक अन्तरक्रियाशीलता**: JavaScript को एक वैकल्पिक एकीकरण के रूप में माना जाता है, जो केवल तब सक्रिय होता है जब समृद्ध क्लाइंट-साइड अन्तरक्रियाशीलता की आवश्यकता होती है।
- **वास्तुशिल्प लचीलापन**: सामग्री-प्रधान पृष्ठों के लिए फ़ाइल-आधारित रूटिंग या उद्यम-स्तरीय API के लिए मॉड्यूल-आधारित रूटिंग के बीच चयन करें।
