# Modular Router (App Router)

**Modular Router** (`src/app/` में स्थित) उद्यम आर्किटेक्चर के लिए डिज़ाइन किया गया है। यह **कंट्रोलर्स**, **मॉड्यूल** और **निर्भरता इंजेक्शन** की प्रणाली पर आधारित है।

अन्य फ़्रेमवर्क के विपरीत, Nexy रूट को परिभाषित करने के लिए क्लास मेथड-आधारित दृष्टिकोण का उपयोग करता है, न कि प्रत्येक मेथड के लिए `@Get` जैसे डेकोरेटर्स का।

## 1. कंट्रोलर (Controllers)

एक **कंट्रोलर** `@Controller()` के साथ सजाया गया एक क्लास है। रूट क्लास के उन **मेथड के नाम** द्वारा परिभाषित किए जाते हैं जो HTTP वर्ब (`get`, `post`, `put`, `delete`, आदि) के अनुरूप होते हैं।

```python
# src/app/blog/blog_controller.py
from nexy.decorators import Controller

@Controller("/blog")
class BlogController:
    def __init__(self, service: BlogService):
        self.service = service

    # एक रूट GET /blog परिभाषित करता है
    async def get(self):
        return await self.service.get_all()

    # एक रूट POST /blog परिभाषित करता है
    async def post(self, data: dict):
        return await self.service.create(data)
```

### कंट्रोलर के मुख्य बिंदु:
- **मेथड के नाम**: HTTP एंडपॉइंट परिभाषित करने के लिए `get`, `post`, `put`, `delete`, `patch` का उपयोग करें।
- **WebSocket**: WebSocket प्रवेश बिंदु परिभाषित करने के लिए `socket` मेथड नाम का उपयोग करें।
- **रूट पैरामीटर**: मेथड पैरामीटर स्वचालित रूप से URL पैरामीटर या अनुरोध बॉडी (FastAPI) से जुड़े होते हैं।
- **स्वचालित इंजेक्शन**: `__init__` कंस्ट्रक्टर को पारित सेवाएं स्वचालित रूप से इंजेक्ट की जाती हैं यदि उन्हें `@Injectable()` के रूप में चिह्नित किया गया हो।

## 2. WebSocket समर्थन

Nexy मॉड्यूलर राउटर के माध्यम से मूल रूप से WebSockets को एकीकृत करता है। बस अपने कंट्रोलर में `socket` नामक एक मेथड परिभाषित करें।

```python
from nexy.decorators import Controller
from fastapi import WebSocket

@Controller("/ws")
class ChatController:
    async def socket(self, websocket: WebSocket):
        await websocket.accept()
        while True:
            data = await websocket.receive_text()
            await websocket.send_text(f"संदेश प्राप्त हुआ: {data}")
```

## 2. मॉड्यूल (Modules)

एक **मॉड्यूल** `@Module()` के साथ सजाया गया एक क्लास है जो कंट्रोलर्स, सेवाओं (providers) और अन्य मॉड्यूल को समूहित करता है।

```python
# src/app/blog/blog_module.py
from nexy.decorators import Module
from .blog_controller import BlogController
from .blog_service import BlogService

@Module(prefix="/v1")
class BlogModule:
    controllers = [BlogController]
    providers = [BlogService]
```

### मॉड्यूल गुण:
- **controllers**: पंजीकृत किए जाने वाले कंट्रोलर क्लास की सूची।
- **providers**: इंजेक्ट करने योग्य बनाए जाने वाले सर्विस क्लास की सूची।
- **imports**: शामिल किए जाने वाले अन्य मॉड्यूल (राउटर) की सूची।

## 3. निर्भरता इंजेक्शन (DI)

किसी सेवा को इंजेक्ट करने योग्य बनाने के लिए, `@Injectable()` डेकोरेटर का उपयोग करें।

```python
# src/app/blog/blog_service.py
from nexy.decorators import Injectable

@Injectable()
class BlogService:
    async def get_all(self):
        return [{"id": 1, "title": "Hello Nexy"}]
```

## 4. रूट मेटाडेटा (वैकल्पिक)

यदि आपको OpenAPI मेटाडेटा (टैग, सारांश, आदि) को अनुकूलित करने की आवश्यकता है, तो `UseRoute` डेकोरेटर का उपयोग करें।

```python
from nexy.decorators import Controller, UseRoute

@Controller("/users")
class UserController:
    @UseRoute(summary="उपयोगकर्ताओं की सूची", tags=["Admin"])
    def get(self):
        return []
```
