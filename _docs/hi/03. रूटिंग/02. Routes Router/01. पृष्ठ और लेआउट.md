# Routes Router: पृष्ठ, API और WebSockets

**Routes Router** (फ़ाइल-आधारित रूटिंग) Nexy का डिफ़ॉल्ट इंजन है। यह आपके एप्लिकेशन के प्रवेश बिंदुओं को स्वचालित रूप से परिभाषित करने के लिए आपकी `src/routes/` निर्देशिका की संरचना का उपयोग करता है, चाहे वे HTML पृष्ठ हों, REST API हों या WebSocket कनेक्शन हों।

## 1. पृष्ठ रेंडरिंग (UI)

`.nexy` और `.mdx` फ़ाइलें स्वचालित रूप से HTML रूट में बदल दी जाती हैं।

- `src/routes/index.nexy` -> `/`
- `src/routes/blog/[slug].mdx` -> `/blog/{slug}`

### लेआउट (Layouts)
किसी फोल्डर में `layout.nexy` फ़ाइल उस फोल्डर और उसके सब-फोल्डर्स के सभी रूट पर स्वचालित रूप से लागू होती है।

## 2. API एंडपॉइंट (.py)

प्योर API एंडपॉइंट (JSON) बनाने के लिए, `src/routes/` में `.py` फ़ाइलों का उपयोग करें। Nexy उन फ़ंक्शन की पहचान करता है जिनका नाम **बड़े अक्षरों** में HTTP वर्ब से मेल खाता है।

```python
# src/routes/api/users.py
from pydantic import BaseModel

class User(BaseModel):
    id: int
    name: str

# एंडपॉइंट GET /api/users
async def GET():
    return [{"id": 1, "name": "Alice"}]

# एंडपॉइंट POST /api/users
async def POST(user: User):
    return {"status": "created", "user": user}
```

## 3. WebSocket समर्थन

Routes Router एक `.py` फ़ाइल में `SOCKET` (बड़े अक्षरों में) नामक फ़ंक्शन के माध्यम से WebSockets का भी समर्थन करता है।

```python
# src/routes/chat.py
from fastapi import WebSocket

async def SOCKET(websocket: WebSocket):
    await websocket.accept()
    await websocket.send_text("Routes Router WebSocket से जुड़ा है!")
    await websocket.close()
```

## 4. निर्भरता प्रबंधन

Nexy Routes Router के लिए एक पदानुक्रमित निर्भरता प्रणाली प्रदान करता है। `src/routes/` की किसी भी सब-डायरेक्टरी में `dependencies.py` फ़ाइल बनाएँ।

```python
# src/routes/admin/dependencies.py
from fastapi import Header, HTTPException

async def verify_token(x_token: str = Header(...)):
    if x_token != "fake-super-secret-token":
        raise HTTPException(status_code=400, detail="X-Token header invalid")

# /admin/ और उसके सब-फोल्डर्स के सभी रूट
# इन निर्भरताओं का उपयोग करेंगे।
dependencies = [verify_token]
```

### निर्भरता की विशेषताएं:
- **विरासत**: `src/routes/` के रूट में परिभाषित निर्भरताएं पूरे एप्लिकेशन पर लागू होती हैं।
- **संचय**: एक सब-फोल्डर अपनी स्वयं की निर्भरताओं को अपने माता-पिता की निर्भरताओं के साथ जोड़ता है।
- **नेटिव FastAPI**: Nexy आंतरिक रूप से `fastapi.Depends` का उपयोग करता है, इसलिए आप FastAPI की सभी सुविधाओं का उपयोग कर सकते हैं।

---

**सलाह**: हल्के एंडपॉइंट के लिए `.py` फ़ाइलों और जटिल सेवाओं के लिए **Modular Router** का उपयोग करें जिन्हें क्लास के बीच उन्नत निर्भरता इंजेक्शन की आवश्यकता होती है।
