# التكوين (`nexyconfig.py`)

تم تصميم Nexy ليعمل بأقل قدر من التكوين (الاصطلاح على التكوين). ومع ذلك، بالنسبة للمشاريع المعقدة، يمكنك تخصيص سلوك إطار العمل عبر ملف `nexyconfig.py` في جذر مشروعك.

## 1. الهيكل الأساسي

يجب أن يحدد الملف فئة `NexyConfig` ترث من `NexyConfigModel`.

```python
from nexy.core.models import NexyConfigModel

class NexyConfig(NexyConfigModel):
    useTitle = "تطبيقي في Nexy"
    useVite = True  # تفعيل تنسيق Vite
```

## 2. خيارات الخادم

- **`usePort`** (`int` ، افتراضي: `3000`): المنفذ الذي سيستمع عليه خادم التطوير والإنتاج.
- **`useHost`** (`str` ، افتراضي: `"0.0.0.0"`): مضيف استماع الخادم.
- **`useDocs`** (`bool` ، افتراضي: `True`): تفعيل أو تعطيل واجهة Swagger UI الخاصة بـ FastAPI.
- **`useDocsUrl`** (`str` ، افتراضي: `"/docs"`): عنوان URL للوصول إلى توثيق Swagger.
- **`useRedocsUrl`** (`str` ، افتراضي: `"/redocs"`): عنوان URL للوصول إلى توثيق ReDoc.

## 3. تنسيق الواجهة الأمامية (Vite)

- **`useVite`** (`bool` ، افتراضي: `False`): تفعيل تكامل Vite. لا غنى عنه إذا كنت تستخدم مكونات React أو Vue أو Svelte أو Solid.js.
- **`useViteDevUrl`** (`str` ، اختياري): عنوان URL مخصص لخادم تطوير Vite (مثال: `http://localhost:5173`).
- **`useAliases`** (`dict` ، افتراضي: `{}`): يحدد أسماء مستعارة للمسارات لاستيراداتك (مثال: `{"@": "src"}`).

## 4. الامتدادات وأطر العمل (useFF)

يسمح Nexy بتوسيع قدراته عبر قائمة `useFF` (Frontend Frameworks).

```python
from nexy.core.models import FFModel

class NexyConfig(NexyConfigModel):
    useFF = [
        FFModel(
            name="react",
            render="render_react", # وظيفة العرض الداخلية
            extension=[".tsx", ".jsx"]
        )
    ]
```

## 5. المحتوى و Markdown

- **`useMarkdownExtensions`** (`list[str]` ، افتراضي: `[]`): قائمة بامتدادات Python-Markdown لتحميلها لعرض ملفات `.mdx` (مثال: `["extra", "codehilite"]`).

## 6. التوجيه والاستبعاد

- **`excludeDirs`** (`list[str]` ، افتراضي: `[]`): قائمة بالأدلة التي يجب تجاهلها أثناء الاكتشاف التلقائي للمسارات (File-Based Routing).
- **`useRouter`** (`APIRouter` ، اختياري): يسمح بحقن مثيل `APIRouter` مخصص عند البدء.

---

**نصيحة**: معظم الخيارات لها قيم افتراضية سليمة. لا تقم بتعديل إلا ما هو ضروري لحالة الاستخدام الخاصة بك.
