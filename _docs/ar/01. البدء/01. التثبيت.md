# التثبيت

Nexy سهل التثبيت والبدء.

## البدء السريع

أسرع طريقة للبدء هي استخدام الأمر `nx init`.

### 1. تثبيت Nexy عبر pip

```bash
pip install nexy
```

### 2. تهيئة المشروع

```bash
nx init my-app
```

سيرشدك المساعد التفاعلي عبر:
- اختيار إطار عمل الواجهة الأمامية المفضل لديك (React أو Vue أو Svelte أو Solid.js).
- تفعيل أو تعطيل Tailwind CSS.
- تكوين هيكل المشروع.

## المتطلبات الأساسية

- **Python 3.10+**
- **Node.js 18+** (مطلوب فقط إذا كنت تستخدم إطارات عمل JS للواجهة الأمامية)

## التشغيل في وضع التطوير

بمجرد تهيئة مشروعك، ابدأ خادم التطوير:

```bash
nx dev
```

سيقوم هذا الأمر بـ:
- بدء خادم الواجهة الخلفية **FastAPI**.
- بدء منسق الواجهة الأمامية **Vite** بالتوازي.
- تفعيل **تبديل الوحدات الساخن (HMR)** لتجربة تطوير سلسة.

---

**نصيحة**: استخدم العلم `--help` مع أي أمر `nx` لرؤية الخيارات المتاحة.
