# مقدمة

Nexy هو إطار عمل Python متكامل (meta-framework) مصمم لتطوير تطبيقات الويب الحديثة. يجمع بين **FastAPI** لمنطق الواجهة الخلفية (backend) و **Vite** للتفاعل في الواجهة الأمامية (frontend)، مما يوفر نظامًا بيئيًا كاملاً للمطورين.

## المبادئ الأساسية

- **أمان الأنواع الموحد (Pydantic)**: حدد نماذج البيانات الخاصة بك مرة واحدة، وسيضمن Nexy التحقق من الأنواع والاتساق عبر النظام بالكامل.
- **الخادم أولاً (SSR)**: يتم عرض التطبيقات على جانب الخادم افتراضيًا للحصول على أفضل أداء وتحسين محركات البحث (SEO).
- **التفاعل الانتقائي**: يتم التعامل مع JavaScript كإضافة اختيارية، يتم تفعيلها فقط عند الحاجة إلى تفاعل غني من جانب العميل.
- **مرونة المعمارية**: اختر بين التوجيه القائم على الملفات للصفحات الغنية بالمحتوى أو التوجيه القائم على الوحدات (modules) لواجهات البرمجة المؤسسية.
