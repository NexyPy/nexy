# Routes Router: الصفحات وواجهات البرمجة و WebSockets

يعد **Routes Router** (التوجيه القائم على الملفات) هو المحرك الافتراضي لـ Nexy. يستخدم هيكل دليل `src/routes/` لتعريف نقاط الدخول لتطبيقك تلقائيًا ، سواء كانت صفحات HTML أو واجهات برمجة تطبيقات REST أو اتصالات WebSocket.

## 1. عرض الصفحات (UI)

يتم تحويل ملفات `.nexy` و `.mdx` تلقائيًا إلى مسارات HTML.

- `src/routes/index.nexy` -> `/`
- `src/routes/blog/[slug].mdx` -> `/blog/{slug}`

### التخطيطات (Layouts)
ملف `layout.nexy` في مجلد ما ينطبق تلقائيًا على جميع المسارات في هذا المجلد ومجلداته الفرعية.

## 2. نقاط نهاية واجهة البرمجة (.py)

لإنشاء نقاط نهاية واجهة برمجة تطبيقات نقية (JSON) ، استخدم ملفات `.py` في `src/routes/`. يحدد Nexy الوظائف التي يتطابق اسمها مع أفعال HTTP بأحرف **كبيرة**.

```python
# src/routes/api/users.py
from pydantic import BaseModel

class User(BaseModel):
    id: int
    name: str

# نقطة نهاية GET /api/users
async def GET():
    return [{"id": 1, "name": "Alice"}]

# نقطة نهاية POST /api/users
async def POST(user: User):
    return {"status": "created", "user": user}
```

## 3. دعم WebSocket

يدعم Routes Router أيضًا الـ WebSockets عبر وظيفة تسمى `SOCKET` (بأحرف كبيرة) في ملف `.py`.

```python
# src/routes/chat.py
from fastapi import WebSocket

async def SOCKET(websocket: WebSocket):
    await websocket.accept()
    await websocket.send_text("متصل بـ Routes Router WebSocket!")
    await websocket.close()
```

## 4. إدارة الاعتماديات

يوفر Nexy نظام اعتمادية هرمي لـ Routes Router. قم بإنشاء ملف `dependencies.py` في أي دليل فرعي لـ `src/routes/`.

```python
# src/routes/admin/dependencies.py
from fastapi import Header, HTTPException

async def verify_token(x_token: str = Header(...)):
    if x_token != "fake-super-secret-token":
        raise HTTPException(status_code=400, detail="X-Token header invalid")

# جميع المسارات في /admin/ ومجلداته الفرعية
# ستستخدم هذه الاعتماديات.
dependencies = [verify_token]
```

### خصائص الاعتماديات:
- **الوراثة**: الاعتماديات المحددة في جذر `src/routes/` تنطبق على التطبيق بالكامل.
- **التراكم**: يراكم المجلد الفرعي اعتمادياته الخاصة مع اعتماديات والديه.
- **FastAPI أصيل**: يستخدم Nexy `fastapi.Depends` داخليًا ، لذا يمكنك استخدام جميع ميزات FastAPI.

---

**نصيحة**: استخدم ملفات `.py` لنقاط النهاية الخفيفة و **Modular Router** للخدمات المعقدة التي تتطلب حقن اعتمادية متقدم بين الفئات.
