# التكامل: React و Vue و Svelte و Solid.js

Nexy غير مرتبط بإطار عمل واجهة أمامية محدد. يمكنك استخدام أي إطار عمل JavaScript رئيسي في صفحاتك التي يعرضها الخادم.

## كيف يعمل

يستخدم Nexy **Vite** كمنسق لتجميع وتنشيط المكونات من جانب العميل.

- **التنسيق عبر Vite**: لاستخدام أطر العمل هذه ، يجب تفعيل تكامل Vite في `nexyconfig.py` عبر `useVite: True`.
- **التنشيط (Hydration)**: يتم عرض مكونات العميل بتنسيق HTML على الخادم (SSR) ثم تنشيطها على العميل لتصبح تفاعلية عبر بنية Vite التحتية.
- **بدون تكوين (Zero Configuration)**: يوفر Nexy تكوين Vite معدًا مسبقًا لكل إطار عمل مدعوم.

## أطر العمل المدعومة

- **React**: دعم كامل لمكونات و hooks React.
- **Vue**: دعم لـ SFC (Single File Components).
- **Svelte**: دعم لمكونات Svelte.
- **Solid.js**: دعم لمكونات Solid.js.

### مثال لمكون React

```tsx
// src/components/Counter.tsx
import { useState } from 'react';

export default function Counter() {
    const [count, setCount] = useState(0);
    return (
        <button onClick={() => setCount(count + 1)} className="px-4 py-2 bg-blue-500 text-white rounded">
            العداد: {count}
        </button>
    );
}
```

### الاستخدام في .nexy

```python
---
from "@/components/Counter.tsx" import Counter
---
<Counter />
```
