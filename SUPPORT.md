# Support

## Community resources

| Resource | Purpose |
|----------|---------|
| [GitHub Discussions](https://github.com/NexyPy/nexy/discussions) | Questions, ideas, show & tell |
| [GitHub Issues](https://github.com/NexyPy/nexy/issues) | Bug reports, feature requests |
| [Documentation](https://nexy.ai/docs) | Official docs, guides, API reference |

## Before asking

1. **Search existing issues** — your question may already be answered.
2. **Check the documentation** at [nexy.ai/docs](https://nexy.ai/docs).
3. **Search Discussions** for similar topics.

## Filing a bug

Open a [bug report](https://github.com/NexyPy/nexy/issues/new?labels=bug&template=01_bug_report.yml)
with:

- Nexy version (`nexy --version`)
- Python version
- Operating system
- Steps to reproduce
- Relevant log output

## Security vulnerabilities

Please **do not** report security issues publicly. Email
[hopcy.forcy@gamil.com](mailto:hopcy.forcy@gamil.com) instead.
See [SECURITY.md](SECURITY.md) for our disclosure policy.

## Feature requests

Open a [feature request](https://github.com/NexyPy/nexy/issues/new?labels=enhancement&template=02_feature_request.yml)
with:

- The problem you're trying to solve
- Your proposed solution
- Any alternatives you've considered

## Commercial support

For commercial support inquiries, please open a
